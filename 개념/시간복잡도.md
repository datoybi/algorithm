## 로그

코테에는 밑이 2인 로그만 나온다. 로그는 해당 수 가 2의 몇 거듭제곱인지를 의미합니다. 아래의 로그만 알아도 무방합니다.

```
// 2를 몇번 곱해야하는지 계산하면 된다.
log2 = 1
log4 = 2
log8 = 3
log16 = 4
log32= 5
```
